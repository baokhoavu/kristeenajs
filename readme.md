# KristeenaJS - Next.js E-Commerce API Intergration

This app was created to help inspire and promote dreams of those breaking into the esports industry using a p2p donation based system. Creating an example using an egirl as the subject for products displayed vertically.

Created using NextJS as the React frontend framework utilizing SCSS to apply styles. The app also uses snipcart's API to perform quick ecommerce services.
